Tarea Programacion II

Recursos HTML Recursos CSS

Descripción

Inicialmente el estudiante deberá registrarse a Udemy y completar los cursos, luego subir a GitHub la página Web realizada durante el curso deberá estar personalizada haciendo una mejora del diseño.

En este curso se aprenderá lo básico de:

  • Etiquetas HTML5.
  • Primeros pasos con CSS3.
  • Layout con Flexbox.
  • Librería Felixblox Grid.

Internet se ha convertido en el mecanismo que utilizamos a diario para trabajar, estudiar, comunicarnos con otras personas, realizar compras, entre otras. Todo esto lo realizamos por medio de la web, la cual funciona por medio de documentos que se enlazan por medio de un servidor y se proyectan en un navegador web.

Por medio de este curso aprenderás todo lo relacionado con los lenguajes HTML y CSS para que puedas crear tus propias páginas web de una manera simple, sin necesidad de contar con conocimientos avanzados para ello.
